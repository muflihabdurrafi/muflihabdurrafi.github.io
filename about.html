Nama saya Muhammad Muflih Abdurrafi, anak ke-4 dari 4 bersaudara. Saya dulu bersekolah international di Saudi Arabia dengan total 6 tahun pendidikan (tidak berturut-turut). Hasilnya, bahasa Inggris saya bisa dibilang lumayan terlatih, baik secara verbal maupun tulisan. Saya pindah dan menetap kembali di Indonesia sejak SMA kelas X pada tahun 2017 hingga lulus pada tahun 2020 dari SMAN 2 Gunung Putri.

Semasa SMA, saya mengambil jurusan IPA karena saya menyukai ilmu matematika. Pada awal SMA saya susah sekali untuk beradaptasi karena saya tidak memiliki relasi satu pun di SMAN 2 Gunung Putri. Meskipun demikian, saya tetap dapat bersosialisasi dan berteman dengan banyak kolega di sekolah. Saya di sekolah dikenal sebagai anak yang ramah dan ambisius, apalagi ketika kelas XII berlangsung. Jarang sekali saya melanggar peraturan karena menurut saya kesuksesan dapat berkembang dari kedisiplinan.

Semenjak remaja saya sudah memegang komputer untuk diri sendiri. Namun, saya belum begitu tertarik pada saat saat itu. Seiring berjalannya waktu, saya mulai membangun minat terhadap dunia teknologi dan ilmu komputer karena berpotensi tinggi pada masa yang akan datang. Saya beberapa kali mencoba untuk meng-coding suatu hal. Beberapa contohnya: membuat website menggunakan HTML, CSS, Javascript; mengonfigurasi suatu server untuk game; dan mempelajari Python. Saya sekarang sedang berusaha untuk menguasai skill pemrograman dengan jangkauan yang lebih luas tentunya.

Dengan demikian, saya mempunyai cita-cita menjadi seorang ahli pemrograman, baik aplikasi maupun perangkat lunak lainnya. Walaupun saya belum mengerti banyak hal mengenai pemrograman, saya akan berjuang sepenuh hati untuk meraihnya.
